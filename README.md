## :bulb:使用说明

### 1. 安装
- `npm` 安装:
  ```bash
  > npm i @2030/admin-v-tab
  ```
- `yarn` 安装:
  ```bash
  > yarn add @2030/admin-v-tab
  ```
- `pnpm` 安装:
  ```bash
  > pnpm add @2030/admin-v-tab
  ```

### 2. 使用
```typescript
import type { ButtonTabProps, ChromeTabProps} from '@2030/admin-v-tab';
import {ButtonTab, ChromeTab} from '@2030/admin-v-tab';
```
